# Prueba E2E
Autor:  Angie Alvarado

Prueba de automatización creada usando Serenity BDD con screenplay

## Prerrequisitos

La herramientas utilizadas para la creación de este proyecto son:

- IntelliJ IDEA
- Java JDK 17
- Gradle 7.6.1




---

## Plugins necesarios en IntelliJ

Se recomienda instalar los siguientes plugins para una mejor experiencia de desarrollo:

- Gherkin
- Cucumber for Java




---

## Ejecución de pruebas

### Ejecutar todos los tests

 ```
 gradle clean test    
 ```     
### Ejecutar pruebas con Log mas detallados     
 ``` 
 gradle clean test --info
 ```
